Requisitos para Adoptar

Para adoptar una mascota, se necesita cumplir con algunos requisitos:

  • Ser mayor de edad.
  • Fotocopias de CI.
  • Fotocopia de luz y agua.
  • Seguimiento mediante videos, fotos o visitas.
  • Tener un comprobante de domicilio.
  • Firmar un contrato de adopción.
  • Realizar un cuestionario de aptitudes.
  • Tener tiempo suficiente para cuidar a la mascota.
  • Tener una vivienda propia o permiso del propietario para tener una mascota si se vive de alquiler.

Obligaciones en el contrato de adopción:

  • Darle una buena alimentación y cuidados veterinarios.
  • Esterilizarlo si tiene la edad suficiente.
  • No abandonarlo.
  • No destinarlo a la cría o reproducción.
  • Notificar cualquier cambio, como pérdida o muerte.
